Para implantação, é preciso:
1) alterar o url na linha 4 do arquivo: webservice_init.php
2) modificar os dados de conexão com o banco de dados das linhas 5 a 8 do arquivo: webservice_init.php

Para fazer uso do serviço de checagem de login, é preciso:
1) modificar o caminho onde o serviço está hospedado. Isso é feito na linha 4 do arquivo: teste.php
2) realizar a passagem de forma dinâmica dos parâmetros nas linhas 7 e 8 do arquivo: teste.php
